Method

Four signal groups. Each one stays visible.

Sift combines evidence instead of relying on a single detector. Strong platform disclosures can move a result quickly. Weaker text or community signals only add context.

01

Platform labels

AI-generated, altered-content, and provenance disclosures shown directly by YouTube or TikTok.

RoleStrong evidence

02

Creator disclosures

Visible captions, descriptions, hashtags, and references to known generation tools.

RoleContextual evidence

03

Watermark matches

Optional low-resolution checks for configured generator watermarks, performed locally in the browser.

RoleSupporting evidence

04

Viewer feedback

AI, human, and unsure votes can support or dispute a classification as feedback accumulates.

RoleSupporting evidence

What the result means

A screening layer, not a verdict.

Sift evaluates production signals. It does not judge whether a claim is true, whether a video is harmful, or whether an AI-assisted edit matters in context.

Read the limitations
Strong signal agreement

Multiple reliable signal groups point in the same direction.

Mixed or limited evidence

The available signals conflict or are too weak for a firm assessment.

No score

Sift leaves the video unscored when there is not enough reliable information.

Installation

Install the beta in four steps.

Chrome requires this manual process because Sift has not been published in the Chrome Web Store.

  1. 01

    Download the versioned ZIP

    Use the release download on this page. Chrome may ask whether you want to keep the file.

  2. 02

    Extract the folder

    Extract sift-extension-v0.4.1 to a stable location. Do not choose the ZIP itself in Chrome.

  3. 03

    Enable Developer mode

    Open chrome://extensions and turn on Developer mode in the upper-right corner.

  4. 04

    Load the extracted build

    Choose Load unpacked, select the folder containing manifest.json, then refresh YouTube or TikTok.

Keep the extracted folder where it is. Moving or deleting it will break the installed beta.

FAQ

Before you install.

Does a Sift result prove that a video is AI-generated?

No. The AI signal score shows which direction the available signals lean, while evidence strength shows how much useful evidence was found. Platform labels can be wrong, disclosures can be missing, and feedback can disagree. Treat the result as a screening signal, not proof.

Does AI-generated mean false or harmful?

No. Sift evaluates production signals, not truthfulness or intent. A synthetic video can be harmless, and a conventional video can still be misleading.

Why can the website not install Sift automatically?

Chrome blocks direct third-party extension installation for normal users. A one-click installation and automatic updates require the Chrome Web Store or an enterprise-managed browser policy.

How do beta updates work?

Updates are manual. Download the newer release, replace the files in your stable Sift folder, then click Reload for Sift on chrome://extensions.

Is the extension open source?

No. The website repository and compiled beta ZIP are public so people can download and inspect them. The development source repository is private.
